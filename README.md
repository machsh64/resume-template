# 简历生成网站

## 使用方式

### **本地服务器**  
   在项目目录执行：
   ```bash
   node server.js
   ```
   浏览器访问：http://localhost:3000

## 功能说明

- **个人信息**：顶部蓝色栏，姓名、性别、电话、邮箱、工作经验、求职意向。
- **教育经历**：学校、学历（本科/专科等）、专业、就读时间与结束时间。
- **专业技能**：支持 Markdown 的多行文本。
- **工作经历**：可多条，每条为公司、岗位、工作时间。
- **项目经历**：可多条，每条为项目名称、角色、时间、项目内容（MD）、项目业绩（MD）。
- **自我评价**：支持 Markdown 的多行文本。
- **预览**：右侧 A4 预览，样式与模板一致（蓝色顶栏、左侧竖条标题）。
- **保存json**：保存此次填写的内容为json文件，供后续使用
- **下载 PDF**：点击「下载 PDF」导出当前预览内容。
